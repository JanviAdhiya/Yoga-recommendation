# Yoga Recommendation App

A full-stack yoga recommendation web application that suggests yoga poses based on user goals such as weight loss, flexibility, and stress relief.

## Features

- User signup and login
- Goal-based yoga recommendations
- Dynamic yoga flow rendering
- MongoDB database integration
- Session tracking
- REST API integration

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

## Note

Authentication and session handling are currently being improved as part of ongoing development.
